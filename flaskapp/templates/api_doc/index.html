In order to use the API you should have and account. The required credentials for the API usage are the "user id" and "authentication token". You can locate this info at the users panel after you login in the site.

How send a SMS

Do a HTTP POST request to this API url "/sms" using the following HTTP POST vars.

  • user_id
    Your user account id, located at user panel.
  • auth_token
    Your user account "authentication token", located at user panel.
  • to
    The phone number where you will send the SMS.
  • body
    The SMS text message to send.
