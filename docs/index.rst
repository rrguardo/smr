Flask-Flaskapp
================

Flaskapp is a basic `Flask`_ web application used as start point for the 
development, including basic configurations for `SQLAlchemy`_, Lazy view load, 
JQuery and Twitter Bootstrap, Babel, cache, default settings, unittests, 
documentation, error pages, custom exceptions, and code structure.

.. _SQLAlchemy: http://www.sqlalchemy.org/
.. _Flask: http://flask.pocoo.org/
